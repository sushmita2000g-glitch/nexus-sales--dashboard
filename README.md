# NEXUS — Sales Command Center

A fully dynamic, real-time Sales Dashboard built using pure HTML5 Canvas API and Vanilla JavaScript. No external chart libraries. No frameworks. Just clean, lightweight code in a single HTML file.

## What It Does

This dashboard visualizes live sales data with animated KPI cards, a 12-month revenue trend line chart, category-wise donut chart, weekly performance bar chart, regional heatmap, scrolling ticker tape, and a top products table. All KPIs auto-update every 4 seconds to simulate live data.

## Tech Stack

HTML5 Canvas API, CSS Grid, Vanilla JavaScript, Google Fonts (Space Mono & Syne)

## Charts Included

Revenue Over Time — smooth bezier curve with gradient fill across 12 months. Sales by Category — animated donut chart with legend. Weekly Performance — grouped bar chart showing actual vs target units sold. Regional Split — horizontal heatmap bars by geography. Sparklines — mini trend lines inside each KPI card.

## How to Run

Download the file `sales_dashboard.html` and open it directly in any modern browser like Chrome or Edge. No installation needed. No server needed. Just open and it works.

## How to Deploy on GitHub Pages

Create a new public repository on GitHub. Upload the `sales_dashboard.html` file. Go to Settings → Pages → select branch `main` → Save. Your live link will be `https://your-username.github.io/repo-name/sales_dashboard.html`
